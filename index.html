Lesson Four

To start, you should know the difference between an "inline element" and a "block element". An "inline" means that you can use one in the same line of text. (ie, 'b', 'u', etc.) A "block" means that it will change the line of text. (ie, 'p', 'h', etc.) Today we'll learn some new tags.

One tag is a 'div' tag. It's basically a 'p' tag. There's also a 'span' tag. That will put two lines of code on the same line of text. This is more useful than it sounds. There's also a 'br' tag. It's self-closing, and it puts a space between lines of text. You may not need it much, but it's good to know.

Now for something interesting. To give your text color, use a tag like this: 'font color= "this is the color"' this is the the text that is colored '/font'. Also,you can make numbered lists and bullet points.

  • To make a bullet point, use a 'ul'tag. Then go to the next line and type an 'li' tag, and type what you want.
  1. For a numbered list, do the same thing, but with 'ol' instead of 'ul'.

To add color to these & other functions, go inside the, in this case, 'li' tag, and add, 'style= "color: a random color"' Anyway, that's all for today. See you next time!

Lesson Five

Now, I don't know if you remember the part where we did a 'body' tag, if not, oh well. Go above the 'body' tag and make a 'head' tag. In the head tag, write:

p {

color:insert color here;

}

Hopefully you have some 'p' tags in your body. Now, save and open your file. Ta-da! Everything in a 'p' tag is that color! Try changing the tag name from p to, well, anything else really. Anything in that tag is now that color!(Note: This will not work if it's done in the 'body' tag.) Today is gonna be a lot of that; welcome to CSS!

Let's continue with text, ok? What if, you could change the size of the text? Without using an 'h' tag? Now you can! Within the curly braces, write : font-size:##px. "px"stands for pixels, so you can make all the text a certain number of pixels! However, all 'h' tags are still in effect. As a note, 15px is the default.

Now, to make the rest of it pretty. Let's start with a simple border.

tag name here {

border: ##px, style type (i.e., dotted, solid, etc), color;

}

You will get that color and style of border! YAY! Note: The border will encircle each of those tags completely and seperately. If you want a full-page border, use "body" as the tag name before the curly brace.

But, what is this? You want the border to be farther away from the text inside it? Try to use the "padding" function, like so:

tag name here {

border: ##px, style type (i.e., dotted, solid, etc), color;

padding:##px;

}

This will seperate the text and the border by the chosen number of pixels. However, if you want to be more specific, try:

tag name here {

border: ##px, style type (i.e., dotted, solid, etc), color;

padding-left:##px;

}

This will cause padding only on the left side of the border. But what if you want the border further from the "walls" of the page, try the "margin" function. It is written exactly like the "padding" function, only with the word "margin" instead. Anyway, that's a border. But what about backgrounds?

To do a background, simply:

tag name here {

background: color;

}

Hooray! A background! Well, I think that's all for today. Oh, and as a note; if you haven't been reading these lessons weekly, you really should be. Every month, I change the theme of this site! So, see you next time for Lesson Six!
